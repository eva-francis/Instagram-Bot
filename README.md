# Instagram Bot
Un bot para Instagram utilizando [Selenium](https://pypi.org/project/selenium/) y [Geckodriver](https://github.com/mozilla/geckodriver).
## Importante
- Por defecto nuestro proxy está definido como Firefox debido a que ha sido el utilizado en las pruebas.
- Los índices de la lista de etiquetas han sido removidos debido a que pertenecían a la versión de prueba.
